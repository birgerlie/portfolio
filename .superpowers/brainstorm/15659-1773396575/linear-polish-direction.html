Elevated Linear

Same principles but stats get individual cards with very subtle fill (3% white) + thin borders. Slightly warmer, more depth. Tables get a surface background too. Feels more like distinct panels.

Key Changes From Current

Typography
  • Negative letter-spacing on values (-0.02em)
  • 11px uppercase labels with 0.05em tracking
  • Font weight 500 instead of light/bold extremes
  • 13px body text (not 14px)
Surfaces & Borders
  • Borders at 6% opacity (not zinc-800)
  • Row dividers at 3-4% opacity
  • Background #0a0a0a (not pure black)
  • Hover states via background opacity shifts
Color
  • Green: #3dd68c (softer than #22c55e)
  • Red: #f76e6e (softer than #f87171)
  • Text hierarchy via opacity (0.4 / 0.65 / 0.9)
  • No saturated accent colors for UI chrome
Spacing
  • Tighter padding (16-18px, not 20-24px)
  • 8-10px row padding (not 12-16px)
  • Smaller border-radius (8-10px, not 12-16px)
  • Consistent 6px label-to-value gap
